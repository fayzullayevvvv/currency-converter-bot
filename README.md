# Currency Converter Bot

A Telegram bot built with **python-telegram-bot** for real-time currency conversion between **USD**, **RUB**, and **UZS**.

The bot is designed with a simple and intuitive user experience in mind: minimal commands and natural text input.

---

## Features
- Real-time currency conversion
- Supported currencies: **USD**, **RUB**, **UZS**
- Natural text input (no command required for conversion)
- Current exchange rates on demand
- Clean and extensible architecture
- Ready for future expansion

---

## How It Works

Users can convert currencies by simply sending a message with an amount and currency.

### Examples

100 USD
100 USD to RUB
25000 UZS to USD
50 RUB to USD
